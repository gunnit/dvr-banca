# DVR Banca — Demo Site

Static demo site for the DVR / ESG banking platform (Niuexa / N2O).

The site is published with **GitHub Pages** from the [`docs/`](docs/) folder:

- [`docs/index.html`](docs/index.html) — landing page
- [`docs/demo_banca_esg.html`](docs/demo_banca_esg.html) — interactive DVR / ESG platform demo
- [`docs/user-stories-dashboard.html`](docs/user-stories-dashboard.html) — user stories dashboard

All pages are self-contained static HTML — no build step required. Edit the files in `docs/` and push to `main` to deploy.

Internal project material (proposals, planning docs, minutes) is intentionally untracked — see `.gitignore`.
